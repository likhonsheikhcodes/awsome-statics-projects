Contribute Your Project

Share your awesome static project with the community. Follow these steps to contribute:

  1. Fork the GitHub repository
  2. Create a new folder for your project in the `projects` directory
  3. Add your project files, including `index.html`, `style.css`, and `thumbnail.png`
  4. Update the `projects.json` file with your project information
  5. Submit a pull request with a brief description of your project
